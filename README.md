# novo-cnpj-php
Biblioteca PHP para validação de CNPJ numérico e alfanumérico conforme o manual oficial do SERPRO.

Biblioteca em PHP para validação de CNPJ numérico e alfanumérico conforme a especificação oficial da Receita Federal do Brasil.

O cálculo dos dígitos verificadores segue o manual técnico publicado pelo SERPRO.

Sem dependências externas.
